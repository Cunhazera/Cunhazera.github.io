Mon Jul 09, 2018 · 84 words
bloghugo

I was a Jekyll user, but I just decided start using Hugo reading a friend’s article.

Any advantage?

Oh, yeah… Installing Hugo is much easier than installing all that Ruby stuff. I did not edit many Jekyll themes, but with Hugo it looks better also.

Disadvantage?

One at least. It’s a little harder to host your Hugo site at Github. The official Hugo documentation did not help me it all, I could only do that reading this post. But after that everything is fine.
